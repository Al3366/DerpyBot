# SteveBot
A Dubtrack Bot using DubAPI

## Requirements
1. Node
2. MongoDB

## Installation
1. `npm install`
2. `mongod` if your database isn't started
3. Create `settings.js` using `example-settings.js`, replacing each variable with your credentials
4. `node index`
